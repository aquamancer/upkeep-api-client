# Upkeep Work Order Fetcher
Fetches all work orders from the Upkeep API, expands ID fields, and converts to .csv

## Usage
1. Clone the repository:
        
        git clone https://github.com/aquamancer/upkeep-api-client
2. Install `pandas`:

        pip install pandas

3. `cd` to the directory containing `main.py`:
        
        cd upkeep-api-client
4. Run the program:

        python main.py
